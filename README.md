# Automation & QA Assessment

## Task 1
Performed manual QA testing on SauceDemo web application.
Identified multiple functional and UX issues with severity classification and root-cause analysis.

## Task 2
Built an n8n workflow using:
- CoinGecko API
- Webhook Trigger
- Data transformation using Code node
- Discord webhook notification

## Workflow Logic
1. Webhook triggers workflow
2. CoinGecko API fetches crypto data
3. Code node filters/transforms data
4. Discord webhook sends crypto brief notification

## Error Handling
Basic workflow validation and execution monitoring were implemented in n8n.

## Tools Used
- n8n
- CoinGecko API
- Discord Webhooks
- SauceDemo
